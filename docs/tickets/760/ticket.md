# Technical Specification ‚Äî Issue #760: Changes to Residential Property Tribunal (new list + name change)

## 1. User Story
**As a** Service manager
**I want to** update the venue name and open justice wording of the Residential Property Tribunal in CaTH, and create a new "Market Rents Weekly Hearing List" hearing list
**So that** the Residential Property Tribunal is up-to-date in CaTH and the new Market Rents list can be published at the same venue.

## 2. Background

The Residential Property Tribunal (RPT) is already implemented in CaTH as a single shared list-type package (`libs/list-types/ftt-rpt-weekly-hearing-list`) that serves five region-specific list type codes (Eastern, London, Midlands, Northern, Southern). All five share one JSON schema, one Excel converter, one PDF generator, one email-summary builder, and one web view template. The venue itself is modelled as **sub-jurisdiction id 24** ("Residential Property Tribunal") in `libs/location/src/location-data.ts`.

This issue requires three coordinated changes:

1. **Rename the venue** from "Residential Property Tribunal" to **"First-tier Tribunal (Property Chamber) - Residential Property Division"** ‚Äî without affecting existing subscriptions.
2. **Add a sixth list type**, "First-tier Tribunal (Residential Property Tribunal): Market Rents Weekly Hearing List", published under the same venue, with an Excel-upload dropdown display name of **"FTT (RPT): Market Rents Weekly Hearing List"**.
3. **Revise the open justice ("Important information") wording** for all RPT regional lists, insert the correct regional office email addresses, and add an extra bold paragraph for the Market Rents list only.

Key existing touch-points identified during research:
- List-type seed data: `libs/list-types/common/src/list-type-data.ts` (lines 337‚Äì391)
- SQL seed: `apps/postgres/prisma/scripts/001_insert_missing_list_types.sql` and sub-jurisdiction links in `003_upsert_sub_jurisdictions_and_list_type_links.sql`
- Venue/sub-jurisdiction: `libs/location/src/location-data.ts:418` (`subJurisdictionId: 24`)
- PDF generator registry: `libs/publication/src/processing/service.ts` (lines 217‚Äì251)
- Excel converter registration: `libs/list-types/ftt-rpt-weekly-hearing-list/src/conversion/ftt-rpt-config.ts`
- Email builder registry: `libs/notifications/src/notification/notification-service.ts` (lines 213‚Äì232)
- Web controller: `apps/web/src/pages/(list-types)/ftt-rpt-weekly-hearing-list/index.ts`
- Locale content: `libs/list-types/ftt-rpt-weekly-hearing-list/src/locales/{en,cy}.ts`

**Note (pre-existing debt):** the RPT `cy.ts` locale file currently holds English text (no real Welsh translation). This work must supply genuine Welsh for all new and revised strings ‚Äî the issue provides the required Welsh translations.
